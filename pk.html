About

This is a Awesome Application created by VitorMM and PaulTheTall

Porting Kit allows a much easier way to play Windows games on OSX

What I do for it

I Help port the games from windows which involves me getting the windows game first createing a wrapper to see if it will run.

If it doesn't run the first time then I have to check the logs to see if it needs a DLL if it does then I add it via wine tricks
